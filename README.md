# 🚀 FLY INVADERS

## 🎮 Giới thiệu

**Fly Invaders** là một game **2D Shooter** được phát triển bằng Unity.
Người chơi điều khiển nhân vật để tiêu diệt kẻ địch và sống sót càng lâu càng tốt.

---

## ⚔️ Gameplay

* 🎯 Độ khó **tăng dần theo điểm số**
* 👾 Số lượng enemy spawn **tăng theo thời gian**
* 🔥 Gameplay nhanh, đòi hỏi phản xạ tốt

---

## 🎮 Điều khiển

* ⬅️➡️ Di chuyển: **Phím mũi tên**
* 🔫 Bắn: **Space**

---

## 🖥️ Nền tảng

* Windows (PC)
* Dự án hiện tại là **source code Unity**

---

## ⚙️ Cách chạy project

1. Mở **Unity Hub**
2. Chọn **Open Project**
3. Trỏ đến thư mục project
4. Nhấn **Play** trong Unity Editor để chạy game
